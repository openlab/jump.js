jump.js - Google maps jQuery plugin Extend with pagination

Using jump.js with Google maps

Download jQuery 1.4.X or higher or use Googles or Microsofts CDN.

<script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false"></script>
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script type="text/javascript" src="PATH_TO_PLUGIN/jump.js"></script>
<script type="text/javascript" src="PATH_TO_PLUGIN/jump.extensions.js"></script>

An example of how to extend the plugin with a simple pagination (see jump.extensions.js).

After extending the plugin, just add some markers and call $('#map_canvas').jump('pagination');

$('#map_canvas').jump().bind('init', function(ev, map) {
	$('#map_canvas').jump('addMarker', {
		'position': '57.7973333,12.0502107', 
		'title': 'Gothenburg'
	}).click(function() {
		$('#map_canvas').jump('openInfoWindow', { 'content': this.title }, this);
	});
	$('#map_canvas').jump('pagination', 'title');
});
